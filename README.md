## Prerequisites

1. Node.js
2. npm
3. Python

## Local Setup

1. Clone the repository:

   ```bash
   git clone https://github.com/joaquinvaldezzz/personacraft.git
   ```

2. Navigate to the project directory:

   ```bash
   cd personacraft
   ```

3. Install the dependencies:

   ```bash
   npm install
   ```

4. Start the development server:

   ```bash
   npm run dev
   ```
